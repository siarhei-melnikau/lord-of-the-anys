# ClimateWatch

A data-driven educational resource on global climate change — built with React + Vite.

## What's inside

- **Hero** with key framing and navigation
- **Stats strip** — live key numbers: temperature anomaly, CO₂ concentration, sea level rise, Arctic ice loss
- **Temperature bar chart** — NASA GISS anomaly data from 1980 to 2024, color-coded by severity
- **1.5°C callout** — contextual section on the Paris Agreement threshold
- **Impacts grid** — 6 topic cards (sea level, heat events, ice melt, precipitation, biodiversity, food security) with risk badges
- **Solutions grid** — 8 decarbonisation pathways
- **Resources** — links to IPCC, NASA, NOAA, IEA, Carbon Brief, Our World in Data

## Tech stack

- [React 19](https://react.dev/)
- [Vite](https://vite.dev/)
- Plain CSS (no UI library)

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
```
